Hi! I'm Norbert.

About Me

As I said, my name is Norbert. I was born in '92 and I'm living in Budapest. I'm studying Photogrammetry and Geoinformatics at the Budapest University of Technology and Economics. In my free time I like to read about new technologies then try 'n' learn them. I'm interested in coding, machine learning (especially deep learning) and in computer vision. Along with these I also like, *NIX systems, IT security and geoinformatics, Augmented Reality. And I would like getting familiar with ERPs (e.g. SAP) in the future.

Interests

Machine learning & computer vision


These are my latest interests. I find these fields really interesting and powerful and imho these will be really importants in the future.
In computer vision I'm reading articles and studies and practicing with the OpenCV library.
I'm kinda newbie in machine learning, but I would like to be an expert. Most of the time I'm using Keras with Tensorflow backend.

Coding


When I'm coding I use Python, JavaScript, BASH, and C++ (of course also use HTML and CSS). At the past I also use Java, C# and Pascal.
From these my all time favourite is Python. I just love it. The package management, the virtual environments and the whole language is awesome. In package management only the NodeJS can beat it imho, but this is another story.

Operating Systems


I grow up on Windows(95 - ) with a few years of C64, but in the last 6 years I'm using linux next to the Windows. Mainly Ubuntu, but also tried Debian, Fedora, Manjaro, Arch, OpenSUSE, deepin, Solus and Mint. I really like to test new distros and desktop enviornments. I found Ubuntu the most stable (I really liked Unity, don't blame me) and the hardware support may better than the others.

The rest


I find photogrammetry and geoinformatics to an interesting field. Always liked to create point cloud and mesh models from photos or working with a lot of data in a GIS (mostly QGIS).
The IT security is just a hobby. I like to reading articles about this topic and keep myself "up-to-date". Using tools and understand how things work is the most important. I like to play with IDA and solve wargames, CTF challanges.
